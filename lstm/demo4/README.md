# LSTM Demo3 说明

本项目是基于TensorFlow实现的LSTM模型演示，采用模块化设计，专注于简单而高效的时序预测实现。

## 1. 项目结构

```
lstm/demo3/
├── main.py          # 主程序入口
├── train.py         # 训练相关代码
├── evaluate.py      # 评估相关代码
├── models.py        # 模型定义
└── data_generator.py # 数据生成和处理
```

## 2. 主要特点

### 2.1 技术栈转换
- 从PyTorch迁移到TensorFlow框架
- 使用TensorFlow的Sequential API构建模型
- 采用Adam优化器和MSE损失函数

### 2.2 模型架构
- 双层LSTM网络结构：
  - 第一层：32个单元，返回序列
  - 第二层：16个单元
- 两层全连接层：
  - 中间层：8个单元，ReLU激活
  - 输出层：单个输出单元

### 2.3 数据处理
- 序列数据生成器
- 灵活的数据预处理流程
- 支持批量数据处理

### 2.4 评估系统
- 模型性能评估
- 预测结果验证
- 测试数据生成

## 3. 使用方法

### 运行程序
```bash
python main.py
```

程序会自动执行以下步骤：
1. 生成训练数据
2. 创建并训练模型
3. 评估模型性能
4. 进行预测演示

## 4. 环境要求

- Python 3.7+
- TensorFlow 2.x
- NumPy
- Matplotlib

## 5. 与Demo2的主要区别

1. 框架切换：从PyTorch转向TensorFlow
2. 架构简化：专注于基础LSTM实现
3. 训练流程：统一的训练和评估过程
4. API使用：采用TensorFlow的高级API
